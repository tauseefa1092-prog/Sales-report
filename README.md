# 📊 Sales Report Dashboard (Power BI)

## 📌 Overview

This project is an interactive **Sales Report Dashboard** built using **Microsoft Power BI**. It provides valuable insights into sales performance through dynamic visualizations, KPIs, and filters, helping users analyze business performance efficiently.

---

## 🚀 Features

* 📈 Sales performance analysis
* 💰 Profit analysis
* 📊 Interactive charts and visualizations
* 🎯 Key Performance Indicators (KPIs)
* 🔍 Dynamic filtering and slicers
* 📅 Time-based sales trends
* 📦 Product/category analysis
* 🌍 Region-wise sales insights (if available)

---

## 🛠️ Built With

* **Microsoft Power BI Desktop**
* **DAX (Data Analysis Expressions)**
* **Power Query**
* **Data Modeling**
